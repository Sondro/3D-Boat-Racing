# [3D Boat Racing Game in Haxe ](https://github.com/Sondro/3D-Boat-Racing): 
[_Featuring Bullet Physics_](https://pybullet.org) 

### Controls:

|   KEY(S)   |          EVENTS         |
| :--------: | :---------------------: |
|     Q      |    1P charge-up left    |
|     E      |    1P charge-up right   |
|     A      |       1P row left       |
|     D      |       1P row right      |
|            |                         |
|     U      |    2P charge-up left    |
|     O      |    2P charge-up right   |
|     J      |       2P row left       |
|     L      |       2P row right      |
